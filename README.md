## 🛠️ Especificações Técnicas (Tech Stack)

Este projeto utiliza tecnologias de ponta para **Visualização de Dados (Data Visualization)** e monitoramento em tempo real:

* **Linguagem:** Python (Backend).
* **Interface:** Streamlit (Frontend).
* **Motor Gráfico:** Plotly (3D Orthographic Projection).
* **Manipulação de Dados:** Pandas.

## 🧠 Arquitetura do Sistema

O **Neural Sentinel** utiliza uma lógica de **Dados Sintéticos (Synthetic Data)** para simular o tráfego de rede global entre metrópoles mundiais. A interface apresenta um globo 3D interativo com logs de eventos de segurança processados em tempo real.
